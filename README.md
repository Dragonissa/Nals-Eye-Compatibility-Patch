# Nals-Eye-Compatibility-Patch

A simple texture patch that replaces wounded, bionic, bandaged, and xenotype eye textures with nothing, improving compatibility with Nal's Facial Animation Textures.

- Missing Eye (Fresh) [Wound]
- Missing Eye [Wound]
- Biotec Eye [Implant]
- Archotec Eye [Implant]
- Red Eyes [Xenotype]
- Gray Eyes [Xenotype]
